URL Shortener API Docs

User Stories:

  1. You submit a valid hostname as a parameter and receive a shortened URL in JSON response.
    API endpoint is GET https://apis-v1-jolav.glitch.me/url/validHostname
  2. If you pass an invalid hostname that doesn't follow the valid www.domain.extension or example.extension format, the JSON response will contain an error instead.
  3. If you visit a shortened URL you will be redirected to the original link.

Example Usage:

  • Show all shortened URLs
    GET https://apis-v1-jolav.glitch.me/url/list
  • Create a new shortened URL
    GET https://apis-v1-jolav.glitch.me/url/new/codetabs.com
  • Use a shortened URL
    GET https://apis-v1-jolav.glitch.me/url/71

Example Output:

    {
      "short": 69 ,
      "original":"codetabs.com",
    }
