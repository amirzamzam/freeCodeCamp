Image Search API Docs

User Stories:

  1. You can get the image URLs, alt text and page urls for a set of images relating to a given search string.
    API endpoint is
    GET /search/search?q=searchThis[&num=number][&offset=number]
  2. You can add a ?offset=(1-100) which result the search should start at.
    You can choose how many responses you get adding parameter ?num=(1-10)
    https://apis-v1-jolav.glitch.me/search/search?q=cats&num=10&offset=5
  3. You can get a list of the most recently submitted search strings.
    https://apis-v1-jolav.glitch.me/search/recent

Example Usage:

  • Show recent searches
    GET https://apis-v1-jolav.glitch.me/search/recent
  • Show the first X results of the search
    GET https://apis-v1-jolav.glitch.me/search/search?q=cats&num=10
