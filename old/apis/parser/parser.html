Header Parser API Docs

User Stories :

  1. You get a JSON response with the IP address, the language (from header Accept-Language ) and the operating system (from header User-Agent ) for your browser.

Example Usage:

  • GET https://apis-v1-jolav.glitch.me/parser/whoami

Example Output:

    {
  "Ip Address": "XXX.XXX.XXX.XXX",
  "Language": "en-US,en;q=0.5",
  "Operative System": 
    "Mozilla/5.0 (Macintosh; Intel Mac OS X 10.12; rv:57.0)
     Gecko/20100101 Firefox/57.0"
  }
